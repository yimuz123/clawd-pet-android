# Clawd Pet Android 🐾

一个运行在安卓手机上的AI桌宠，基于Claude Code，悬浮窗常驻屏幕，随时聊天陪伴。

## 主要功能

- **悬浮窗常驻**：透明悬浮窗小螃蟹趴在屏幕上，随时聊天，可识别屏幕内容
- **AI聊天**：双击桌宠打开聊天框，基于Claude Code，sessionKey登录（暂时只支持官方Claude Code登录，free用户无CC模式）
- **模型切换**：支持Opus / Sonnet / Haiku，可以自定义系统提示词设定人设
- **语音回复**：支持TTS语音（暂时只支持MiniMax API）
- **看屏幕**：发送"看一下屏幕"触发截屏，桌宠会分析你的屏幕内容
- **智能操作**：帮你定手机闹钟、设计时器、加日历、打开APP
- **10+种表情**：根据当前心情，在聊天时做出对应动作
- **触摸互动**：双击聊天、三击戳一戳、长按拖动、靠边贴边探头
- **对话记录**：自动保存聊天记录，支持多轮对话和历史管理

## 安装

1. 下载本仓库的 `clawd-pet.apk`
2. 传到手机上安装（可能需要开启"允许安装未知来源应用"）
3. 打开app

## 使用教程

1. 装好apk打开，先填sessionKey（登录claude.ai拿cookie里的sessionKey，以sk开头）
2. 点"启动桌宠"，小螃蟹就出来了
3. 双击它打开聊天输入框，三击戳一戳
4. 长按可以拖着走，靠边会自动贴边探头
5. 设置里可以调大小（80-300dp）、换模型、写人设、开TTS语音
6. 看屏幕的话需要授权截屏权限
7. 发送"看一下屏幕"触发截屏

### 获取SessionKey

1. 手机或电脑浏览器打开 [claude.ai](https://claude.ai) 并登录
2. 按F12打开开发者工具（手机可以用Kiwi浏览器）
3. 找到Application（应用）→ Cookies → `https://claude.ai`
4. 复制 `sessionKey` 的值（以 `sk-ant-` 开头）

### 互动方式

| 操作 | 效果 |
|------|------|
| 双击桌宠 | 打开聊天框 |
| 三击桌宠 | 戳一戳，触发反应 |
| 长按拖动 | 移动桌宠位置 |
| 拖到屏幕边缘 | 贴边探头模式 |
| 贴边时单击 | 取消贴边 |

## 注意事项

- 需要开悬浮窗权限，安卓设置里搜"悬浮窗"或"显示在其他应用上方"并打开
- sessionKey过期了重新登claude.ai拿一个，为sk开头
- 聊天消耗的是你claude.ai账号的额度
- 设置里可以调节桌宠大小（80-300dp）
- 对话记录保存在手机本地
